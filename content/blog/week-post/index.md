## Что было сделано на этой неделе

На этой неделе я начал работу над своим индивидуальным проектом в рамках курса «Операционные системы». Основная задача первого этапа — развернуть персональный сайт с помощью генератора статических сайтов Hugo и опубликовать его на GitHub Pages.

### Основные шаги

В ходе работы были выполнены следующие задачи:

1. **Установка необходимого ПО** — установил Hugo и Git на свой компьютер.
2. **Создание репозитория** — создал репозиторий blog на GitHub на основе шаблона HugoBlox Academic CV.
3. **Настройка GitHub Pages** — создал репозиторий Jamderov.github.io и настроил автоматический деплой через GitHub Actions.
4. **Настройка baseURL** — указал правильный путь `/blog/` в конфигурации Hugo для корректной работы CSS и JS.
5. **Добавление личных данных** — заполнил информацию о себе: имя, биографию, интересы и образование.

### Впечатления

Работа с Hugo оказалась достаточно простой и понятной. Шаблон Academic CV предоставляет удобную структуру для академического портфолио. GitHub Pages позволяет бесплатно размещать статические сайты, что очень удобно для студентов.

### Планы на следующую неделю

На следующей неделе планирую:

- Загрузить свою фотографию на сайт
- Написать пост на тему управления версиями с помощью Git
- Продолжить настройку и персонализацию сайта
